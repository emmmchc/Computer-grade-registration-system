## Java Web 课程设计

### 题目

**计算机等级考试系统**

#### 第一天

1. 搭建运行环境 IDEA2020、Tomcat7.0.9 、JDK1.8 、MYSQL5.7、普通javaweb项目
2. 搜集资料（Github、百度、知网）
3. 选择技术栈 jsp 、jstl 、EL表达式语言 、ajax 、jdbc 、javabean
4. 分层设计 util、filter、servlet、service、dao、entity
5. 角色设计 学生 院系管理员 系统管理员  创建相关实体表
6. 界面设计 div + css 布局 为每个角色创建相应导航界面
7. 为3个角色写最简单的CRUD(先不考虑冗余代码，优化问题，实现优先)，采用之前练习过的Druid作为连接池
8. 登录、注册界面设计，利用课上的表单验证例子做一些简单的表单验证
9. 选一个角色，登录后台，写个简单的利用ajax修改密码功能 



#### 第二天

1. 实现学生注册功能
2. 设计考生报名流程
3. 考生报名界面设计
4. 考生需要填写的信息设计
5. 创建报名表



#### 第三天

1. 实现报名信息存储

2. 实现院系管理员对报名信息的分页、编辑、删除、查询

3. 实现系统管理员对院系管理员、注册学生、报名学生的管理

4. 代码整合

   

#### 第四天

1. 修复一些bug
2. 增加准考证号、考场、座位
3. 代码优化
4. 写报告



#### 第五天

1. 修改报告
2. 等待答辩
